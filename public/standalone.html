🎯 抽獎原理

本系統採用密碼學安全隨機數產生器確保每次抽獎結果完全隨機且無法預測。

✅ 每位候選人都有相同的中獎機率

✅ 無法人為操控抽獎結果

✅ 每次抽獎都是獨立事件

📋 抽獎步驟

  1. 載入所有候選人資料到記憶體陣列中
  2. 使用 Fisher-Yates 洗牌演算法建立陣列副本
  3. 從陣列最後一個元素開始，逐一往前處理
  4. 對於位置 i，從 0 到 i 中隨機選擇一個位置 j
  5. 交換位置 i 和位置 j 的元素
  6. 重複步驟4-5直到處理完所有元素
  7. 從完全隨機排列的陣列中取出前 N 位作為得獎者
  8. 回傳得獎者名單和抽獎時間戳記

const data = await response.json(); if (!response.ok) { throw new Error(data.error || "抽獎失敗"); } // 儲存結果並顯示 currentWinners = data.winners; drawTimestamp = data.timestamp; displayResults(data); } catch (error) { console.error("抽獎錯誤:", error); showError(error.message); } finally { // 恢復按鈕狀態 drawBtn.disabled = false; loading.style.display = "none"; } } // ===== 後端抽獎核心演算法 (實際執行) ===== // Fisher-Yates 洗牌演算法 (此網站實際使用的演算法) function shuffleArray(array) { const shuffled = [...array]; for (let i = shuffled.length - 1; i > 0; i--) { // 使用 Math.random() 產生隨機索引 const j = Math.floor(Math.random() * (i + 1)); // 交換元素位置 [shuffled[i], shuffled[j]] = [shuffled[j], shuffled[i]]; } return shuffled; } // API路由：進行抽獎 (Express.js 後端實際程式碼) app.post("/api/draw", (req, res) => { try { const { count = 50 } = req.body; // 驗證抽獎人數 if (count > allParticipants.length) { return res.status(400).json({ error: `抽獎人數 (${count}) 不能超過總參與者人數 (${allParticipants.length})`, }); } // 使用Fisher-Yates演算法隨機打亂陣列，然後取前N個 const shuffled = shuffleArray(allParticipants); const winners = shuffled.slice(0, count); // 回傳抽獎結果 res.json({ success: true, totalParticipants: allParticipants.length, drawCount: count, winners: winners, timestamp: new Date().toLocaleString("zh-TW"), }); } catch (error) { console.error("抽獎錯誤:", error); res.status(500).json({ error: "抽獎過程發生錯誤" }); } }); // 載入參與者資料 (CSV處理) function loadParticipants(filePath) { return new Promise((resolve, reject) => { const participants = []; fs.createReadStream(filePath, { encoding: "utf8" }) .pipe(csv()) .on("data", (row) => { // 處理CSV資料並建立參與者物件 const nameKey = Object.keys(row).find( (key) => key.trim().toUpperCase() === "NAME" ); // 其他欄位處理... if (nameKey && row[nameKey] && row[nameKey].trim() !== "") { participants.push({ name: row[nameKey].trim(), bindIdKey ? (row[bindIdKey] || "").trim() : "", id: idKey ? (row[idKey] || "").trim() : "", idAll: idAllKey ? (row[idAllKey] || "").trim() : "", phone: phoneKey ? (row[phoneKey] || "無").trim() : "無", rawData: row, }); } }) .on("end", () => { resolve(participants); }) .on("error", (error) => { reject(error); }); }); } // ===== Fisher-Yates 演算法原理說明 ===== // // 1. 從陣列最後一個元素開始往前處理 // 2. 對於位置 i，從 0 到 i 中隨機選擇一個位置 j // 3. 交換位置 i 和位置 j 的元素 // 4. 重複步驟2-3直到處理完所有元素 // 5. 最終得到完全隨機排列的陣列 // // ===== 此演算法的優點 ===== // // ✅ 時間複雜度 O(n)，效率高 // ✅ 空間複雜度 O(1)，記憶體使用少 // ✅ 每種排列出現的機率相等 // ✅ 數學上已證明的公平性 // ✅ 廣泛使用且經過驗證 // // ===== 為什麼選擇 Fisher-Yates 而非其他方法？ ===== // // 相比其他隨機抽取方法： // - 比多次隨機選取更公平（避免重複選取的複雜性） // - 比排序隨機數更有效率 // - 比建立隨機索引陣列更簡潔 // - 是學術界和工業界公認的標準演算法
